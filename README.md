TaskNest - Task Management Web App

A full-stack task management app built for the Web Design and Development Final Project.

Tech Stack
- Design: Figma
- Frontend: React JS, HTML5, CSS3, JavaScript, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Version Control: Git & GitHub

 Architecture — Separation of Concerns
| Layer | Technology | Responsibility |
| Frontend | React JS | UI, user interaction, display data |
| Backend | Node.js + Express | Business logic, API routes |
| Database | MongoDB Atlas | Store and retrieve data |

Features
-  Create, view, update and delete tasks
-  Task status: Pending / Completed
-  Task priority: Low / Medium / High
-  Due date for each task
-  Created date shown on every task
-  Search tasks by title or description
-  Filter tasks by status
-  Dashboard showing total, pending and completed counts
-  Responsive design for mobile, tablet and desktop
